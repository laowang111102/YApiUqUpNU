# Javaweb图书管理系统 java464

## 技术栈与环境

### 技术栈
- Javaweb
- JSP
- Html
- Maven

### 开发环境
- IDE: IntelliJ IDEA
- 数据库: MySQL
- 服务器: Apache Tomcat

## 功能模块

### 用户后台功能
1. 个人资料管理
   - 展示与修改用户资料
2. 借书信息管理
   - 查询借阅记录
   - 查询图书借阅状态
3. 还书信息管理
   - 查看还书记录
   - 执行还书操作
4. 我的收藏管理
   - 管理收藏的图书信息

### 管理员后台功能
1. 普通管理员管理
   - 修改密码
   - 账号管理
2. 用户管理
   - 添加用户信息
   - 管理用户信息
3. 类型信息管理
   - 添加图书类别
   - 查询图书类别
4. 图书信息管理
   - 添加图书信息
   - 查询图书信息
5. 借书信息管理
   - 审核借书请求
   - 查询借书记录
6. 还书信息管理
   - 审核还书操作
   - 查询还书记录
7. 系统管理
   - 设置公告信息

### 用户前台功能
- 具体功能请参考项目图片介绍
